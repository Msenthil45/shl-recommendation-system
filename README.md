# AI Assessment Recommender System

A Flask-based web application that recommends the most relevant SHL assessment URLs based on a job description or user query using TF-IDF and Cosine Similarity.

## Features
- **Natural Language Search:** Input job descriptions to find matching assessments.
- **Machine Learning Backend:** Uses Scikit-Learn's TF-IDF Vectorizer to understand query context.
- **Web Interface:** Clean UI built with Flask and HTML/CSS.

- ## Prerequisites
- Python 3.8+
- The dataset file: `Gen_AI Dataset.xlsx`

- pip install flask pandas scikit-learn openpyxl

- python app.py
